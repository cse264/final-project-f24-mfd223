Spring Backend - API Documentation

Welcome to the backend service. The server is running successfully.

Available REST APIs:

  • User Registration:

    Endpoint: POST /api/users/register

    Request Body:

    {
    "username": "testuser",
    "password": "password123",
    "paidUser": true
}

    Response: 200 OK or relevant error messages

  • User Login:

    Endpoint: POST /api/users/login

    Request Body:

    {
    "username": "testuser",
    "password": "password123"
}

    Response: 200 OK with user details or relevant error messages

  • Weather Information:

    Endpoint: GET /api/weather

    Query Parameters:

    • city (required): Name of the city
    • paidUser (required): true or false

    Response: Weather details or relevant error messages

Frontend Developer Instructions:

Use the above endpoints in your frontend to send and retrieve data. For example:

  • For user registration, send a POST request to /api/users/register with the required JSON body.
  • For fetching weather data, use a GET request to /api/weather with the query parameters.

Contact the backend team if you need further assistance.
